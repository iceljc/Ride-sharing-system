Steps to install and run the programs.
(This application is based on MySQL)

1. Create a database, named 'pickmeup', on your local computer.

2. Import the data to the database. The data is in the folder \pickup\Dataset.

3. Set the right username('root' or defined by yourself), password (defined by yourself) and database name('pickmeup') in \pickup\condb.php.

4. After completing these steps, open your browser and input localhost/pickup/login.php. Then you should see the login page of the system.
